# 🚀 AI Resume Builder with ATS Analyzer

A modern, real-time resume builder designed to create **professional, ATS-optimized resumes** with an integrated **smart analysis engine**.

This project focuses on combining clean UI, real-time feedback, and intelligent suggestions — similar to what you’d expect from a premium SaaS tool.

---

https://resume-builder-ecru-eight.vercel.app/

## ✨ Key Features

### 📄 Resume Builder

* Real-time resume generation as you type
* Clean, recruiter-friendly layout
* Structured sections:

  * Personal Information
  * Professional Summary
  * Skills
  * Experience
  * Projects
  * Education

---

### ⚡ Floating ATS Analyzer (Highlight Feature)

* Floating action button for quick access
* Input job role and job description
* Generates:

  * ATS Score (0–100)
  * Keyword match percentage
  * Missing keywords
  * Improvement suggestions

---

### 🧠 Smart Enhancements

* “Improve Summary” button for better phrasing
* Action-oriented content suggestions
* Better readability and impact

---

### 🎨 UI/UX

* Premium dark sidebar + clean resume canvas
* Responsive layout
* Smooth user interactions
* Designed to feel like a real product, not a basic form

---

### 💾 Data Persistence

* Auto-save using localStorage
* Resume data remains even after refresh

---

## 🛠️ Tech Stack

* HTML5 (Semantic structure)
* CSS3 (Flexbox + Grid)

No frameworks used — built from scratch for full control and performance.

## 🚀 How to Run

### Option 1 (Recommended)

Use Live Server in VS Code:

1. Install Live Server extension
2. Right-click `index.html`
3. Click **"Open with Live Server"**

---

### Option 2

Open directly in browser:

```
index.html
```

(Works best with a local server due to ES modules)
